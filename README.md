🎉 Resep Fidder

This is my first Resep Fidder web application, which allows users to explore recipes from around the world. Users can search for recipes by name and view details such as ingredients, cooking steps, category, and photos.

✨ Key Features

🔍 Recipe Search: Quickly find recipes by name.
🍳 Detailed Recipe Information: View ingredients, cooking steps, category, and photos.
🌐 TheMealDB API Integration: Fetch real-time recipe data from TheMealDB.
💾 Responsive Design: Modern, lightweight interface that works seamlessly on all devices.

🛠️ Technologies Used

Frontend: Vite, React, Tailwind CSS, React Modules
API: TheMealDB API
Tools: VS Code, Git, GitHub
